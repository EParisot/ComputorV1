# ComputorV1
42 Project : calculator and 1st, 2nd degree solver

## requirements
Python3
tkinter (optional)

## Usage
```
python3 computor.py [--gui] ["calcul/equation string"]

or 

python3 computor.py [--gui]
> calcul/equation string
```

## Limitations
1 var max
2nd degree max

## Tests
```
python3 tests.py [-A] or [0][1][2]

```
